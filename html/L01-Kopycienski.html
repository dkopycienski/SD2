This lab introduces us to basic HTML tags

The <DOCTYPE> tag is used for notifying your browser about what type of document to expect, in this case an HTML document.

The <CHARSET> tag chooses the encoding type for the document, in this case, the UTF-8 character set because it contains all of the characters and symbols in the world. This is the one recomended for web development.
The <TITLE> tag specifies the web page title. This is a text title which is displayed in the browser's title or the page's tab.

The <blockquote> tag specifies a section that is quoted from another source.

The <b> tag bolds the text

The <i> tag italicizes the text

The <hr> tag defines a theme break in the text and is shown with a horizontal line

The <strong> tag marks a part of text as very important and bolds it

The <em> tag marks text meant to be emphasized and italicises it
