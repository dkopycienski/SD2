DATABASE COMMANDS

Commands Syntax Description
Create database CREATE DATABASE databasename; used to create a new SQL database.
Drop database DROP DATABASE databasename; used to delete an existing SQL database.
Show databases SHOW DATABASES; used to show existing databases in MySQL
Use database USE databasename Does not require semicolon, allows you to access a database
Set Privileges GRANT _ ON _ TO __; assigns privileges and roles to MySQL user accounts and roles
Remove Privileges REVOKE _ ON _ FROM __; enables system administrators to revoke privileges and roles, which can be revoked from user accounts and roles

TABLE COLUMN COMMANDS

Commands Syntax Description
Create table CREATE TABLE table_name ( column1 datatype, column2 datatype, column3 datatype, .... ); used to create a new table in a database
Drop table DROP TABLE table_name; used to drop an existing table in a database.
Alter table ALTER TABLE table_name ADD column_name datatype; used to add, delete, or modify columns in an existing table.

COLUMN ATTRIBUTES and MODIFIERS

Commands Syntax Description
Constraints CREATE TABLE table_name ( column1 datatype constraint, column2 datatype constraint, column3 datatype constraint, .... ); used to specify rules for data in a table.
NOT NULL NOT NULL Ensures that a column cannot have a NULL value
UNIQUE UNIQUE Ensures that all values in a column are different
PRIMARY KEY PRIMARY KEY A combination of a NOT NULL and UNIQUE. Uniquely identifies each row in a table
FOREIGN KEY FOREIGN KEY Prevents actions that would destroy links between tables
DEFAULT City varchar(255) DEFAULT 'Sandnes' used to set a default value for a column
CHECK CHECK (Age>=18) =, >, or < used to limit the value range that can be placed in a column
Data Types CREATE TABLE table_name ( column1 datatype, column2 datatype, column3 datatype, .... ); defines what value the column can hold
Text CHAR(size), VARCHAR(size), TEXT(size), ENUM(val1, val2, val3, ...), SET(val1, val2, val3, ...) Various string data types, char is for a specific amount of characters, varchar is a range of character with the maximum at the set value
Numeric INT(size), INTEGER(size), DECIMAL(size, d), BOOL Differing data types to store number data, for BOOL, 0 is false and nonzero is true
Date and Time DATE, DATETIME(fsp), YEAR YYYY-MM-DD, YYYY-MM-DD hh:mm:ss, A year in four-digit format, CURDATE(), CURRENT_TIMESTAMP()

TABLE DATA COMMANDS

Commands Syntax Description
Select SELECT column1, column2, ... FROM table_name; used to select data from a database.
Where Clause SELECT column1, column2, ... FROM table_name WHERE condition; used to filter records.
ORDER BY Keyword SELECT column1, column2, ... FROM table_name ORDER BY column1, column2, ... ASC|DESC; used to sort the result-set in ascending or descending order. ASC by default, this keyword must be the last command
INSERT INTO Statement INSERT INTO table_name (column1, column2, column3, ...) VALUES (value1, value2, value3, ...); used to insert new records in a table. Can specify column names and values OR if you are adding values for all columns, you do not need to specify column names
UPDATE Statement UPDATE table_name SET column1 = value1, column2 = value2, ... WHERE condition; used to modify the existing records in a table. Always specify the WHERE condition, could update all data in table
