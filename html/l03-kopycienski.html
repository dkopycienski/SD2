HTML Commands

Item Description
Headers <h1> - <h6>
<b> Bolds the text
Head Tags <head>
<title>
<meta charset="utf-8">
<html lang="en">
<!DOCTYPE html>
<i> italicizes the text
<blockquote> specifies a section that is quoted from another source
<hr> defines a theme break in the text and is shown with a horizontal line
<em> marks text meant to be emphasized and italicises it
<strong> marks a part of text as very important and bolds it
<ul> specifies an unordered list
Seperate items using <li>
<ol> specifies an ordered list
Seperate items using <li>
<dl> marks a term and defintion
mark terms using <dt> and mark definitions using <dd>
<a href= marks a link, after equals sign put link, close the bracket, and then type desriptive text for the link
<img> Links to an image location and name and displays it on the page

CSS Styles

Style Description
background-color
border-collapse: collapse single line border
color text color
text sizes in pixels (px)
border weight in pixels
font:family text font
Font:size in pixels
Font:weight bold or normal
margin-left:auto and margin-right:auto centers item
padding-left and padding-right spaces in table cells
Text-align:center Centers the text in the middle of the page
tr:nth-child(even) {background-color: silver;} Sets even numbered rows of a table a different color for easier readabilty

Website Design Best Practices

Avoid using too many different fonts
Avoid using too many different colors
Have lots of white space
Minimize text
Show, don't tell
Simplify the navigation
Optimize your design for mobile use
